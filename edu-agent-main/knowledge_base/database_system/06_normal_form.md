# 范式

## 知识点概述
范式用于评价关系模式设计质量，常见目标是减少冗余和异常。

## 核心概念
- 1NF：属性不可再分。
- 2NF：消除非主属性对候选码的部分函数依赖。
- 3NF：消除非主属性对候选码的传递函数依赖。
- BCNF：每个非平凡依赖的决定因素都是超码。

## 常见易错点
- 只有复合候选码才可能出现部分依赖。
- 满足 BCNF 一定满足 3NF，但满足 3NF 不一定满足 BCNF。

## 示例
R(A,B,C), F={A->B, B->C}，A 是候选码，B->C 形成传递依赖，不满足 3NF。

## 练习题
说明 3NF 与 BCNF 的判断差异。
